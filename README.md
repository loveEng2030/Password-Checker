# Password Checker and Generator

## Overview
This Java project allows users to **check the strength of their passwords** or **generate strong passwords**.  

The program evaluates passwords based on:
- Length (minimum 8 characters)  
- Uppercase letters  
- Lowercase letters  
- Numbers  
- Special symbols  

It can also generate strong passwords containing a mix of letters, numbers, and symbols.

---

## Features

### 1. Password Strength Checker
- Users can enter a password.  
- The system evaluates its strength on a scale from 1 to 10.  

### 2. Password Generator
- Users specify the desired password length (minimum 8 characters).  
- The program generates a strong random password with at least:
  - 1 uppercase letter  
  - 1 lowercase letter  
  - 1 number  
  - 1 symbol  

---

## Classes

- **Main.java** – Program entry point, menu interface for users.  
- **PasswordChecker.java** – Checks password strength.  
- **PasswordGenerator.java** – Generates strong random passwords.
